## **🗂️ Commit Convention**

#### **📌 Type**
| Tag      | 뜻            |
|-----------|-----------------|
| Feat      | 새로운 기능 추가 및 수정   |
| Fix       | 버그 수정  |
| Docs  | 문서 추가 및 수정 |
| Test      | 테스트 코드 추가 및 수정    |
| Refactor     | 코드 리팩토링 |
| Rename     | 파일 및 폴더명 수정 |
